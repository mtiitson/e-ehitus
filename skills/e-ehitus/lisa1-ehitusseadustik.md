# Ehitusseadustik Lisa 1 (muudetud sõnastuses)

Tabel ehitusteatise, ehitusprojekti ja ehitusloa kohustuslikkuse kohta.

Source: https://www.riigiteataja.ee/aktilisa/1180/3202/6007/Lisa1_28032026.pdf

Legend:
- **Puudub** = no document needed
- **Ehitusteatis** = construction notice only
- **Ehitusteatis ja ehitusprojekt** = construction notice with construction project
- **Ehitusluba** = construction permit required

Columns: Püstitamine/Rajamine = new construction; Ümberehitamine = renovation; Laiendamine = extension; Osa asendamine = partial replacement; Lammutamine = demolition.

For hoone (buildings), the "Ümberehitamine" column is split: first sub-column = ainult tehnosüsteemi osa (technical systems only), second = ümberehitamine või laiendamine kuni 33%.
